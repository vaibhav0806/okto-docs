### Methods Overview

| Methods                            | Description                                          |                 
|------------------------------------|------------------------------------------------------|
| [`showWidgetSheet`](#show-widget-sheet)   | Displays the widget sheet                            |
| [`closeBottomSheet`](#close-bottomsheet)       | Closes the currently opened widget sheet                      |

`showWidgetSheet()`<a href="https://github.com/okto-hq/okto-sdk-react-native/blob/769a2f6e4faec12e45f9b5577ee1b35334e02c7b/src/index.tsx#L49" target="_blank" rel="noopener noreferrer" style= {{ textDecoration: "none" }}> <IIcon icon="lucide:external-link" height="17" style={{ display: 'inline-block', verticalAlign: 'middle' }} /></a> displays the widget modal.

### Parameters

There are no parameters for this function.

### Response

<Callout title="Success Response">

| Field Name | Type                | Description                                 | 
|------------|---------------------|---------------------------------------------|
| `result`   | `void`              | No successful response returned             | 

</Callout>
