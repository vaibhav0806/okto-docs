## Transfer tokens

<sub><i>async</i></sub> `transferTokens(data)`<a href="https://github.com/okto-hq/okto-sdk-react-native/blob/769a2f6e4faec12e45f9b5577ee1b35334e02c7b/src/index.tsx#L120" target="_blank" rel="noopener noreferrer" style= {{ textDecoration: "none" }}> <IIcon icon="lucide:external-link" height="17" style={{ display: 'inline-block', verticalAlign: 'middle' }} /></a> transfers tokens based on the provided data.

### Parameters

| Parameter   | Type            | Description                                     | 
|-------------|-----------------|-------------------------------------------------|
| `data`     | [`TransferTokens`](/api-docs#model/transfertokens) | Data being provided for transferring the tokens | 

### Response

<Callout title="Success Response">

| Field Name | Type                          | Description                                      | 
|------------|-------------------------------|--------------------------------------------------|
| `result`   | [`Promise<TransferTokensData>`](/api-docs#model/transfertokensdata) | Returns information about the tokens transferred | 

</Callout>



<Accordions>
<Accordion title="Example">
<Tabs items={['Typescript']}>
  <Tab value="Typescript">
    ```typescript
        import { useOkto, type OktoContextType} from 'okto-sdk-react';
 
        const { transferTokens } = useOkto() as OktoContextType;
 
        const networkName = "POLYGON_TESTNET"
        const tokenAddress = "x2f7b97837f2d14ba2ed3a4b2282e259126a9b848"
        const quantity = "1"
        const recipientAddress = "0x0FC096A53343C68c1086e626661A4bb31486A9e2"
 
        transferTokens({
            network_name: networkName,
            token_address: tokenAddress,
            recipient_address: recipientAddress,
            quantity: quantity
        }).then((result) => {
                console.log('Transfer success', result);
        })
        .catch((error) => {
            console.log('Transfer error', error);
        });
    ```
  </Tab>
</Tabs>
</Accordion>
</Accordions>

## Fetch order history

<sub><i>async</i></sub> `orderHistory(query)`<a href="https://github.com/okto-hq/okto-sdk-react-native/blob/769a2f6e4faec12e45f9b5577ee1b35334e02c7b/src/index.tsx#L100" target="_blank" rel="noopener noreferrer" style= {{ textDecoration: "none" }}> <IIcon icon="lucide:external-link" height="17" style={{ display: 'inline-block', verticalAlign: 'middle' }} /></a> fetches the order history based on the provided query.

### Parameters

| Parameter   | Type                  | Description                      | 
|-------------|-----------------------|----------------------------------|
| `query`     | [`Partial<OrderQuery>`](/api-docs#model/orderquery) | Query for fetching order history | 

### Response

<Callout title="Success Response">

| Field Name | Type                 | Description                                 | 
|------------|----------------------|---------------------------------------------|
| `result`   | [`Promise<OrderData>`](/api-docs#model/orderdata) | Returns information about the order history | 

</Callout>

<Accordions>
<Accordion title="Example">
<Tabs items={['Typescript']}>
  <Tab value="Typescript">
    ```typescript
        import { useOkto, type OktoContextType, type Order} from 'okto-sdk-react';
 
        const { orderHistory } = useOkto() as OktoContextType;
 
        orderHistory({order_id: orderId})
        .then((orders) => {
            const order = orders.find((order: any) => order.order_id === orderId);
            if(order) console.log(order)
        })
        .catch((error) => {
            console.error(`error:`, error);
        });
    ```
