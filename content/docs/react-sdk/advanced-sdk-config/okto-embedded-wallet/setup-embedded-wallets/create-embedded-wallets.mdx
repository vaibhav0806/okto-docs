## Create a new wallet for current user

<sub><i>async</i></sub> `createWallet()`<a href="https://github.com/okto-hq/okto-sdk-react/blob/main/src/OktoProvider.tsx#L340" target="_blank" rel="noopener noreferrer" style= {{ textDecoration: "none" }}> <IIcon icon="lucide:external-link" height="17" style={{ display: 'inline-block', verticalAlign: 'middle' }} /></a> retrieves all the wallets of the currently authenticated user.

### Parameters

There are no parameters for this function.

<Callout title="Success Response">

| Field Name | Type                  | Description                                   |

| `result`   | [`Promise<WalletData>`](/api-docs#model/walletdata) | Returns the newly created wallet for the user |
