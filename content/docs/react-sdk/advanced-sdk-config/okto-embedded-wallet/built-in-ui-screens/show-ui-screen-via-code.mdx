|------------------------------------|------------------------------------------------------|
| [`showWidgetModal`](#show-modal)   | Displays the widget modal                            |
| [`closeModal`](#close-modal)       | Closes the currently open modal                      |

## Set theme

`setTheme(theme)`<a href="https://github.com/okto-hq/okto-sdk-react/blob/main/src/OktoProvider.tsx#L492" target="_blank" rel="noopener noreferrer" style= {{ textDecoration: "none" }}> <IIcon icon="lucide:external-link" height="17" style={{ display: 'inline-block', verticalAlign: 'middle' }} /></a> sets the UI theme based on the provided theme object.

### Parameters

| Parameter   | Type                 | Description                                               | 
|-------------|----------------------|-----------------------------------------------------------|
| `theme  `   | [`Partial<Theme>`](/api-docs#model/theme)     | Theme provided to set                                     | 

### Response

<Callout title="Success Response">

| Field Name | Type                | Description                                 | 
|------------|---------------------|---------------------------------------------|
| `result`   | `void`             | No successful response returned              | 

</Callout>
