## Get all supported networks 

<sub><i>async</i></sub> `getSupportedNetworks()`<a href="https://github.com/okto-hq/okto-sdk-react/blob/main/src/OktoProvider.tsx#L303" target="_blank" rel="noopener noreferrer" style={{ textDecoration: 'none', display: 'inline-block', verticalAlign: 'middle' }}> <IIcon icon="lucide:external-link" height="17" style={{ display: 'inline-block', verticalAlign: 'middle' }} />
</a> retrieves all the whitelisted networks on your API Key.

### Parameters

There are no parameters for this function.

### Response

<Callout title="Success Response">

| Field Name | Type | Description |
|------------|-----------------------------|------------------------------------------|
| `result`   | [`Promise<NetworkData>`](/api-docs#model/networkdata) | Returns list of whitelisted networks |

</Callout>

                console.log(result)
            })
