This section displays a comprehensive list of all orders placed by users through your application on Okto.

### **Features:**

* Orders are listed in descending order of completion datetime.  
* Search functionality allows you to find specific orders by:  
  * User ID  
  * Order ID  
  * Bulk Order ID  
  * Transaction hash

### **Filtering Options:**

You can filter the order history by:

* Order type  
* Network  
* Status
