The User Log section provides a detailed list of all users who have interacted with your application through Okto. It gives you a snapshot of user activity and growth:

* **Total Users**: View the total number of users associated with your vendor ID.  
* **User Growth**: Track the increase in users over the last month.  
* **New Users**: Monitor newly onboarded users via your vendor ID and track daily increases.

### **Features:**

* Users are listed in descending order of their last login datetime.  
* Search functionality allows you to find specific users by:  
  * User ID  
  * Wallet address  
  * User's login details
