Sandbox is a testing environment that mimics the production environment but with limitations. It allows you to:
    - Experiment with Okto features  
    - Test your integration in both testnet and mainnet  
    - Simulate various scenarios without affecting real data or incurring costs

**Sandbox Limitations**

1. **Rate Limits**:  
    - No rate limits  
2. **Number of Projects**:  
    - Only 1 project will enabled per user as of now  
3. **Feature Restrictions**:  
    - No restrictions
